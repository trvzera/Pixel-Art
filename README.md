# Pixel Art

Editor de pixel art no navegador: grade 10×10, clique para pintar cada quadrado. Projeto simples em HTML, CSS e JavaScript.

## Code Club

Desenvolvido em um dia de treinamento do **[Code Club](https://www.codeclub.org.uk/)** — iniciativa internacional que ensina programação de forma prática, principalmente para jovens, por meio de projetos curtos e acessíveis. A atividade base era montar a grade e colorir os pixels; depois do encontro, acrescentei melhorias por conta própria.

## Funcionalidades

- **Paleta de cores** — tons pré-definidos e seletor de cor livre (`input type="color"`)
- **Tema dinâmico** — a cor escolhida atualiza fundo, título e botões da interface
- **Reiniciar** — limpa o desenho e volta todos os pixels ao estado inicial
- **Visualizar** — alterna para o modo sem bordas entre os pixels, para ver o resultado como uma imagem contínua (e volta ao modo edição quando quiser)

## Como usar

Abra `index.html` no navegador ou use um servidor local. Clique em uma cor, depois nos pixels da grade para desenhar.
